# Panduan Mendapatkan API Key Apilogy (Harga Pangan)

Untuk menghubungkan AgriSensa dengan data harga pangan real-time, Anda perlu mendapatkan API Key dari platform Apilogy (Telkom Indonesia). Berikut langkah-langkahnya:

## 1. Registrasi Akun
1.  Kunjungi portal Apilogy: **[https://apilogy.id](https://apilogy.id)**
2.  Klik tombol **"Daftar"** atau **"Sign Up"** di pojok kanan atas.
3.  Isi formulir pendaftaran dengan data diri Anda.
4.  Lakukan verifikasi email jika diminta.

## 2. Berlangganan API (Subscribe)
1.  Setelah login, masuk ke menu **"API Product"** atau **"Marketplace"**.
2.  Cari produk bernama **"Harga Pangan"** atau **"Food Price"** (biasanya disediakan oleh BigView).
3.  Klik produk tersebut untuk melihat detailnya.
4.  Pilih paket yang tersedia (cari yang **Free Tier** / Gratis untuk percobaan).
5.  Klik **"Subscribe"** atau **"Berlangganan"**.

## 3. Mendapatkan API Key
1.  Setelah berhasil subscribe, masuk ke menu **"My Apps"** atau **"Dashboard"**.
2.  Buat aplikasi baru (Create New App) jika belum ada, beri nama misalnya "AgriSensa".
3.  Di dalam detail aplikasi tersebut, Anda akan melihat bagian **"Credentials"** atau **"Keys"**.
4.  Salin **Consumer Key** (ini biasanya yang digunakan sebagai API Key) atau **Access Token**.

## 4. Integrasi ke AgriSensa
Setelah Anda mendapatkan API Key, silakan kirimkan kepada saya dengan format berikut di chat:

```text
API Key Apilogy saya: [TEMPORKAN_KEY_DISINI]
```

---
**Catatan:**
Jika Anda kesulitan menemukan API Key yang spesifik, Anda bisa menyalin semua informasi kredensial yang muncul (Consumer Key, Consumer Secret) dan saya akan membantu memilahnya.
