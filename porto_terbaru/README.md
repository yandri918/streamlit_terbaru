- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection

### Viewing the Portfolio

1. **Visit the live site**: [https://porto-terbaru.vercel.app](https://porto-terbaru.vercel.app)

2. **Explore projects**:
   - Scroll through the case studies
   - Click "View Technical Details" for deep dives
   - Visit live demos to interact with applications
   - Check GitHub repositories for source code

3. **Navigate sections**:
   - Hero - Introduction and CV download
   - Projects - Portfolio showcase
   - Case Studies - Detailed project breakdowns
   - Skills - Interactive radar chart
   - Tech Stack Timeline - Career progression
   - About Me - Background and expertise
   - Hire Me - Services and availability
   - Contact - Get in touch

### Local Development

If you want to run the portfolio locally:

```bash
# Clone the repository
git clone https://github.com/yandri918/porto_terbaru.git
cd porto_terbaru

# Open in browser
# Simply open index.html in your browser
# Or use a local server:
python -m http.server 8000
# Then visit http://localhost:8000
```

## 🚢 Deployment

### Vercel (Current)

The portfolio is automatically deployed to Vercel:

1. **Connected Repository**: `https://github.com/yandri918/porto_terbaru`
2. **Auto-Deploy**: Pushes to `main` branch trigger deployment
3. **Production URL**: `https://porto-terbaru.vercel.app`

### Manual Deployment

To deploy to other platforms:

#### Netlify
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Deploy
netlify deploy --prod
```

#### GitHub Pages
```bash
# Enable GitHub Pages in repository settings
# Select main branch as source
# Site will be available at: https://yandri918.github.io/porto_terbaru
```

#### Custom Server
```bash

# Ensure index.html is in the root directory
# Configure server to serve static files
```

## 🎨 Customization

### Colors

Edit CSS variables in `styles.css`:

```css
:root {
    --primary-color: #8b5cf6;
    --secondary-color: #3b82f6;
    --accent-color: #10b981;
    --bg-dark: #0f172a;
    --bg-darker: #020617;
}
```

### Content

- **Personal Info**: Edit `index.html` - Hero section
- **Projects**: Edit `index.html` - Projects and Case Studies sections
- **Skills**: Edit `premium-script.js` - Radar chart data
- **Technical Details**: Edit `technical-details.js` - Modal content

### Adding New Projects

1. Add project card in `index.html` (Projects section)
2. Add case study in `index.html` (Case Studies section)
3. Add technical details in `technical-details.js`
4. Update README.md with project information
